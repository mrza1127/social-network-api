# Social Network API

## Description
Backend for a social network application

## Technologies Used
- JavaScript
- NodeJS
- Express.js
- MongoDB

## Installation

- You must run `npm install` at the root of the project
- Run `mongod` to start a mongoose server
- Run `npm start` to begin running application
